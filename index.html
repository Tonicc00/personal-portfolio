FreeCodeCamp - Responsive Web Design

Online location

In this Responsive Web Design Certification I learned the languages that developers use to build webpages: HTML (Hypertext Markup Language) for content, and CSS (Cascading Style Sheets) for design.

First I build a cat photo app to learn the basics of HTML and CSS. Later I learned modern techniques like CSS variables by building a penguin, and best practices for accessibility by building a quiz site.
